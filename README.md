# Telecom Site Tracker

A local-first tracker for telecom site progress updates, blockers, approvals, and field proof.
Data is stored in browser IndexedDB and no backend is required.

## Run

```bash
npm install
npm run dev
```

The dev server runs at http://localhost:5185.

## Verify

```bash
npm run build
```

On Windows PowerShell, use `npm.cmd run build` if script execution is blocked.
